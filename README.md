# React Portfolio | Azaria Francis

A modern, responsive personal portfolio built from the ground up using **React**, **Tailwind CSS**, and **Framer Motion** for sleek animations. This project showcases my journey from a Computer Science graduate to a professional developer.

## Key Features
*   **Dynamic Project Filtering:** Browse projects by category (Java, Web, Mobile).
*   **Custom Typewriter Effect:** Engaging hero section animations built as a reusable component.
*   **Fully Responsive Design:** Optimized for seamless viewing on mobile, tablet, and desktop.
*   **Contact Integration:** Functional contact form ready for EmailJS or backend connection.
*   **Client-Side Routing:** Managed with `react-router-dom` for a smooth Single Page Application (SPA) experience.

## Tech Stack
*   **Frontend:** React.js, Tailwind CSS
*   **Navigation:** React Router 6
*   **Animations:** CSS3 / Custom React Components
*   **Icons/Assets:** Heroicons, custom image assets

## Getting Started

### Prerequisites
*   Node.js (v18 or higher recommended)
*   npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com
   
2. Navigate to the project folder:
    cd react-portfolio
    
3. Install dependencies:
    npm install
    
4. Start the development server:
    npm run dev
